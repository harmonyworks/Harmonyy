Mobile App Technical Issues

Understanding what went wrong with Harmony mobile

Technical Challenges

Developing and maintaining the Harmony mobile application presented several significant technical challenges:

Platform Fragmentation

Incompatibilities across thousands of Android/iOS versions and devices

Resource Consumption

Excessive battery drain and memory usage issues

Update Delays

App store review processes prevented timely bug fixes

Data Syncing Issues

Inability to reliably sync data with web platform

After months of attempting to resolve these issues, we determined that the resources required to maintain an acceptable mobile experience would be better invested in improving our web platform.

Data Transfer Notice: Due to the technical issues with our mobile infrastructure, we are unable to transfer your mobile app data to the web platform.

Our Path Forward

By focusing exclusively on our web platform, we can:

  • Deliver new features more rapidly
  • Ensure a consistent experience across all devices
  • Reduce maintenance overhead
  • Focus resources on core functionality
  • Provide instant updates without app store delays

Frequently Asked Questions

Find answers about the Harmony service transition

Common Questions

Why was the mobile app discontinued?
The mobile app was discontinued due to persistent technical issues that affected user experience. Despite our best efforts, we couldn't resolve all the bugs while maintaining performance standards across diverse mobile devices.
Can I transfer my playlists to the web version?
Unfortunately, due to technical limitations in our mobile infrastructure, playlist and favorite data cannot be transferred to the web platform. You'll need to create new playlists on the web version.
How do I access Harmony on my phone now?
Visit harmonyworks.github.io/Harmony in your mobile browser. The web app is fully optimized for mobile use and can be added to your home screen for app-like access.
Does the web version have the same features?
The web version has all the core features of the mobile app plus additional capabilities. We're continuously adding new functionality to the web platform.
How often is the web platform updated?
We deploy updates to the web platform weekly, sometimes even daily. Without app store review processes, we can fix issues and add features much faster.
